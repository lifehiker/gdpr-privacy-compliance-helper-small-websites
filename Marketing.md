## Target Users

| Persona | Description |
|---|---|
| Solo founders / indie hackers | Running SaaS landing pages, want basic compliance without legal overhead |
| Freelance web developers | Auditing client sites, need evidence capture and a shareable report |
| Small agencies | Managing 1–20 client websites, value bulk workflow and PDF exports |

**Common thread:** Not lawyers. Don't want to become compliance experts. Need to know what's missing, how serious it is, and what to fix first.

---

## Actual App Capabilities (for accurate marketing)

- Automated website privacy scanner — detects missing policy links, cookie banners, trackers, analytics scripts, forms, third-party embeds
- Manual compliance checklist — data retention, legal basis, DSAR process, processor list, and other non-detectable items
- Remediation task tracker — turn findings into actionable fix items
- Evidence capture — attach notes and URLs to each checklist item
- Compliance score (0–100) — single number summary of posture
- PDF export (Pro/Agency plans) — shareable audit report for clients
- S3 file uploads for evidence attachments
- Billing: Free / Pro ($19/mo) / Agency ($59/mo) via Stripe

**Priority: High**
Agencies bring 5–50 client sites each. A referral or white-label program turns one agency signup into a compounding revenue source. Even a simple referral discount at launch captures this leverage.

**Priority: High**
A public scanner page (no login required) is the strongest acquisition wedge. Users get immediate value, see their score, then hit a paywall for the full checklist + task tracker. This also generates SEO-friendly content (scan results as shareable links).
